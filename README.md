# Project on Demography of Students and Their Preferences

This project was developed as part of a high school assignment in Kathmandu, Nepal. The objective of the project was to analyze the demographic characteristics of students and understand their preferences based on collected data.

The study focuses on factors such as age, gender, educational background, and personal interests, aiming to identify patterns and trends among students. The project demonstrates basic data collection, analysis, and interpretation skills.

## Key Features
- Data collection from students
- Analysis of demographic information
- Identification of trends and preferences
- Simple data visualization and reporting

## Tools Used
- Python (Jupyter Notebook)
- Basic data analysis techniques
